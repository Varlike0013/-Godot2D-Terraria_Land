# <h1 id="0"># -Godot2D-Terraria\_Land</h1>

# <anchor id="2"></anchor>

# <p id="3">creater\_by: VarlikeAlert</p>

# <p id="4">一款基于 <strong>Godot Engine</strong> 开发的2D泰拉瑞亚同人游戏，融合养成、策略、建造与休闲元素，还原经典泰拉瑞亚玩法的同时，加入专属特色内容，适合各类喜欢2D像素风、探索建造类游戏的玩家。</p>

# <h2 id="5">🎮 游戏类型</h2>

# <p id="6">2D · 角色养成 · 策略战斗 · 建造创造 · 休闲探索</p>

# <h2 id="7">✨ 核心特色</h2>

# <ul id="8">

# &#x20; <li id="9"><strong>角色养成</strong>：提升角色属性、解锁技能，根据自身玩法习惯打造专属战斗/生存角色，适配不同战斗场景与建造需求。</li>

# &#x20; <li id="10"><strong>建造设施</strong>：自由搭建基地、防御工事、功能建筑，收集材料解锁更多建造样式，打造独一无二的专属家园。</li>

# &#x20; <li id="11"><strong>小怪清理</strong>：探索地图过程中遭遇各类小怪，合理运用角色能力与道具，清理障碍、收集掉落物资。</li>

# &#x20; <li id="12"><strong>Boss击杀</strong>：解锁不同难度Boss，考验角色实力与策略搭配，击败Boss可获得稀有材料与强力奖励，推进游戏进程。</li>

# </ul>

# <anchor id="13"></anchor>

# <h2 id="14">🛠️ 开发环境</h2>

# <p id="15">引擎：Godot Engine（推荐4.6.1及以上版本，Standard版即可）</p>

# <p id="16">开发语言：GDScript</p>

# <h2 id="17">📥 如何运行</h2>

# <ol id="18">

# &#x20; <li id="19">下载Godot Engine（推荐4.6.2）：<a href="https://godotengine.org/download/">Godot官方下载页</a></li>

# &#x20; <li id="20">克隆本仓库：<code>git clone git@github.com:Varlike0013/-Godot2D-Terraria\_Land.git</code></li>

# &#x20; <li id="21">打开Godot Engine，导入本项目文件夹，点击运行即可启动游戏。</li>

# </ol>
